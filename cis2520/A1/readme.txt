How to run the program:
- Compile the program by typeing make in the command line
- Run the program by typeing ./a.out in the command line
- A list of options will appear
- Press 1 and enter to print out all permutations of 'C' 'A' 'R' 'B' 'O' and 'N'
- Press 2 and enter to be brought to the ackermann function 
		- enter two numbers seperated by a space and the output of the function will be outputed
- Press 3 and enter to be brought to the newton function
		- enter the number you wish to find the square of
- Press 4 and enter to be brought to the recursive newton function
		- enter the number you wish to find the square of
- Press 5 to exit the program
